# 나다움 모바일 웹앱 프로토타입

React, Vite, Tailwind CSS 기반의 청년 고립 및 고독감 해소용 생활 커뮤니티 플랫폼 프로토타입입니다.

## 실행

```bash
npm install
npm run dev
```

## 포함 화면

- 로그인
- 온보딩 / 청년 인증
- 메인 홈
- 커뮤니티
- 모임
- 청년지원
- 마이페이지

모든 데이터는 서버 없이 mock data로 동작합니다. 청년 인증, 모임 참여, 지원 저장, 커뮤니티 공감, 심리테스트 결과가 화면 상태에 반영됩니다.

## 주요 구성

- 390px~430px 모바일 앱 중심 UI
- 홈, 커뮤니티, 모임, 지원, 마이 하단 탭
- 나다움 지도, 걸음수 챌린지, 오늘 추천
- 온라인/오프라인 모임 구분
- 성장 기록과 배지 중심 마이페이지
